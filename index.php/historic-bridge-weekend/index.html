Sponsoring Event
Historic Bridge Weekend
  A week before his death in 2012, Bill was planning to start a new event.  A historic landmark or Historic Bridge Weekend.  We, the K8CJQ Memorial Club/Mid West DX Club, are sponsoring the Historic Bridge Weekend. 

4th annual event will be held on July 15-17,  2016.

 Operation on any amateur radio frequencies is permitted.

If you are operating from a historic bridge please email (tw.terrie@gmail.com) the callsign being used, name of the bridge and location including state or country, so that we can get you posted to the website.

If you operated in 2015, Please contact tw.terrie@gmail.com so we can update the website - Thank you
